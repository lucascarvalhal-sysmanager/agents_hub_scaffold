# Catálogo de Componentes

Repositório centralizado de componentes reutilizáveis para agentes.

## Estrutura

- **tools/** - Ferramentas individuais para agentes
- **skills/** - Combinações de tools com instruções específicas
- **callbacks/** - Callbacks reutilizáveis (before/after model, agent, tool)

## Como usar

Cada componente possui:
- `spec.yaml` - Especificação e metadados
- `requirements.txt` - Dependências Python
- `readme.md` - Documentação de uso
